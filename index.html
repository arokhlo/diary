با این قابلیت، هر رویداد جدید مستقیماً در فایل index.html روی GitHub ذخیره می‌شود و از هر دستگاهی قابل دسترس خواهد بود.

چطور Personal Access Token بسازید:
۱. به github.com/settings/tokens/new بروید
۲. یک نام دلخواه بدهید (مثلاً: diary-sync)
۳. تیک گزینه repo را بزنید
۴. روی Generate token کلیک کنید
۵. Token را کپی کرده اینجا paste کنید
